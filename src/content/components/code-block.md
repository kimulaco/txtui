## Basic

```
┌──────────────────────────────┐
│ const x = 42;                │
│ console.log(x);              │
└──────────────────────────────┘
```

## With Language

```
┌─ javascript ─────────────────┐
│ const x = 42;                │
│ console.log(x);              │
└──────────────────────────────┘
```

## With Line Numbers

```
┌──────────────────────────────┐
│ 1 │ const x = 42;            │
│ 2 │ const y = 10;            │
│ 3 │ console.log(x + y);      │
└──────────────────────────────┘
```
