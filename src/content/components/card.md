## Basic

```
┌──────────────────────────────┐
│                              │
│ Card Title                   │
│                              │
│ This is the card body with   │

│                              │

```

## With Image

```
┌──────────────────────────────┐
│ ┌──────────────────────────┐ │
│ │                          │ │
│ │       ┌──────────┐       │ │
│ │       │  IMAGE   │       │ │
│ │       └──────────┘       │ │
│ │                          │ │
│ └──────────────────────────┘ │
│                              │
│ Card Title                   │
│                              │
│ A card with an image area    │
│ above the content section.   │
│                              │

```
