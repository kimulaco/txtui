title: Tree View
description: Hierarchical tree structure for displaying nested data like file systems.
tags:

---

## Basic

```
src/
├── components/
│   ├── Header.tsx
│   ├── Footer.tsx
│   └── Button.tsx
├── pages/
│   ├── index.tsx
│   └── about.tsx
└── styles/
    └── global.css
```

## With Icons

```
# my-project
├── # src
│   ├── # components
│   │   ├── @ Header.tsx
│   │   └── @ Button.tsx
│   └── # pages
│       └── @ index.tsx
├── @ package.json
└── @ README.md
```

## Expanded and Collapsed

```
v src/
  v components/
      Header.tsx
      Button.tsx
  > pages/
  > styles/
  package.json
  README.md
```
