## Basic

```
Name:     Alice
Email:    alice@example.com
Role:     Admin
Status:   Active
```

## With Borders

```
┌──────────┬───────────────────┐
│ Name     │ Alice             │
├──────────┼───────────────────┤
│ Email    │ alice@example.com │
├──────────┼───────────────────┤
│ Role     │ Admin             │
├──────────┼───────────────────┤
│ Status   │ Active            │
└──────────┴───────────────────┘
```

## Inline

```
Name: Alice  |  Email: alice@example.com  |  Role: Admin
```
