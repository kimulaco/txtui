  - layout

## Collapsed

```
┌──────────────────────────────┐
│ > Section 1                  │
├──────────────────────────────┤
│ > Section 2                  │
├──────────────────────────────┤
│ > Section 3                  │
└──────────────────────────────┘
```

## Expanded

```
┌──────────────────────────────┐
│ v Section 1                  │
│                              │
│   This is the content of     │
│   section 1. It can contain  │
│   any text or information.   │
│                              │
├──────────────────────────────┤
│ > Section 2                  │
├──────────────────────────────┤
│ > Section 3                  │
└──────────────────────────────┘
```

## Multiple Open

```
┌──────────────────────────────┐
│ v Section 1                  │
│                              │
│   Content for section 1.     │
│                              │
├──────────────────────────────┤
│ > Section 2                  │
├──────────────────────────────┤
│ v Section 3                  │
│                              │
│   Content for section 3.     │
│                              │
└──────────────────────────────┘
```
